# RAG Angular UI

This is the Angular frontend for the RAG project.

- Provides a user interface for document ingestion, embedding, and chat.
- Connects to a Node.js/Express backend (optionally).
- Features a status/landing page and a multi-step workflow.

## Features
- Status & ReadMe Doc view (static, no backend call required)
- Enter the Application (main app UI)
